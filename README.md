# vue3-pure-bs

- 大屏采用 scale 进行自适应
- 以 1920 \* 1080 作为标准尺寸开发
- 以屏幕高度作为基础进行缩放，同时窗体尺寸做等比缩放
- 最终效果是以设计稿开发的高度占满，宽度等比缩放的自适应大屏

## 场景

### 场景一

> 总分辨率为固定为 9600\*2160，分为左--中--右三块屏幕拼接而成， 不会适配过大过小的屏幕

- 左屏分辨率：2880\*2160
- 中屏分辨率：3840\*2160
- 右屏分辨率：2880\*2160

### 场景二

> 基础分辨率为 1920\*1080，要求向上向下能适配 16:9 的屏幕

- 1920\*1080
- 1280\*720
- 2560\*1440
- 3840\*2160
- 7480\*3240

### 场景三

> 多个 2880\*2160 拼接

- 2880\*2160

### 场景四

- 6000\*2160
